1NF GUIDE

Goal: Every cell must be ATOMIC (contain exactly one piece of information).


Rule 1: No Lists
If a cell contains "Kitchen, Hallway", it violates 1NF. You must split these into two separate rows.


Rule 2: Unique Identification
Once data is atomic, you must identify a combination of columns (Composite Key) that makes every row unique.
