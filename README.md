## Local hosting
You can run the project locally from the `EduBytes_Samuel` folder.

1. Open a terminal in the project folder.
2. Run `python -m http.server 8000` or `python3 -m http.server 8000`.
3. Open `http://localhost:8000` in your browser.

Alternatively, double-click `start-local-host.bat` to start the server automatically.

For a different port, run:

```bat
start-local-host.bat 8080
```
